WM Zoom

Plugin jQuery para visualização de imagens em alta definição

Opções de configuração do plugin jQuery WM Zoom

O objeto config

O objeto config é responsável pelas configurações do plugin. Exemplo: 

				  			$('.my-zoom').WMZoom({
								config : {
									inner : false
								}
							});

Opções do objeto config

Segue lista de opções para o objeto de configurações do plugin:

Atributo Descrição
stageW int - Define a largura da area de visualização da imagem grande. Caso a opção inner for true stageW será ignorado. Default 500
stageH int - Define a altura da area de visualização da imagem grande. Caso a opção inner for true stageH será ignorado. Default 400
inner boolean - Define se o zoom será em uma area exterana em relação a imagem default ou se será interna. Se true será interna. Default false
position string - Define em que posição externa a area de visualização será mostrado. Opções ['top', 'right', 'bottom', 'left']. Caso a opção inner for true position será ignorado. Default 'right'
margin int - Define o espacamento entre a imagem default e o box de visualização. Caso a opção inner for true margin será ignorado Default true

Exemplo:

							$('.my-zoom').WMZoom({
								config : {
				                    stageW : 500,
				                    stageH : 400,
				                    inner  : false,
				                    position : 'right', // [top, right, bottom, left]
				                    margin : 10
				                }
							});


html data attribute

Segue lista de parâmetros obrigátorios que devem ser passados via atributos html:

Atributo Descrição
data-hight-src Define a url da imagem em alta resolução que será visualizada no zoom
data-loader-src Define a url do gif animado que será exibido enquanto a imagem grande carrega no zoom
